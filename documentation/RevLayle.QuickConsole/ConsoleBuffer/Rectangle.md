# ConsoleBuffer.Rectangle method

Draws a filled rectangle in the console buffer. If the width and/or height go outside the boundary of the console buffer, the rectangle is simply truncated at the corresponding edges.

```csharp
public void Rectangle(int x, int y, int width, int height, ConsoleBufferCell cell)
```

| parameter | description |
| --- | --- |
| x | X position of the top-left corner of the rectangle |
| y | Y position of the top-left corner of the rectangle |
| width | Width of the rectangle. |
| height | Height of the rectangle |
| cell | The ConsoleBufferCell value to fill the rectangle with |
