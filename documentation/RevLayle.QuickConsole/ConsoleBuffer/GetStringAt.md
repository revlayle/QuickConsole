# ConsoleBuffer.GetStringAt method

Gets a text string from the console buffer.

```csharp
public string GetStringAt(int x, int y, int length)
```

| parameter | description |
| --- | --- |
| x | X position to get text from |
| y | Y position to get text from |
| length | How many cells to read. If length go outside the boundary of the console buffer, read length is truncated. |

## Return Value

Text string read from the buffer, sans any color information. Out-of-bounds coordinates return an empty string. 0 or negative length values return an empty string. Returned strings with null character values are trimmed from the string.
