# ConsoleBuffer.Copy method

Get a copy of a rectangular area of the console buffer

```csharp
public IConsoleBuffer Copy(int x, int y, int width, int height)
```

| parameter | description |
| --- | --- |
| x | X position of the top-left rectangular area to get a copy |
| y | Y position of the top-left rectangular area to get a copy |
| width | Width of the area to copy |
| height | Height of the area to copy |

## Return Value

A new instance implementing IConsoleBuffer with the copies cells and the width and height provided. Note, if the width and/or height go beyond the console buffer's boundaries, you will get a smaller buffer copy.

## Exceptions

| exception | condition |
| --- | --- |
| ArgumentException | X and/or Y are out-of-bounds, or width and height are not both greater than zero. |
