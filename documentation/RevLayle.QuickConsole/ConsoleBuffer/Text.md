# ConsoleBuffer.Text method (1 of 3)

Write text to the buffer at location x/y. If x or y is out-of-bounds, nothing is written. If the length of the text go beyond the width of the console buffer, the text will be truncated. The console's CurrentForegroundColor and CurrentBackgroundColor colors are used for the cell colors of the text.
