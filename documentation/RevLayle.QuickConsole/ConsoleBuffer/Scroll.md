# ConsoleBuffer.Scroll method

Scrolls a buffer horizontally and/or vertically.

```csharp
public void Scroll(int xd, int yd)
```

| parameter | description |
| --- | --- |
| xd | The X-delta to scroll horizontally. Negative values will scroll left. Positive values scroll right. An X-delta of zero will not do any horizontal scrolling. |
| yd | The Y-delta to scroll vertically. Negative values will scroll up. Positive values scroll down. A Y-delta of zero will not do any vertical scrolling. |
